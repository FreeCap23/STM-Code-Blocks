Parses a command and stores the command ID and argument in the given pointers' locations.

This function works with the following format: <COMMAND_ID:COMMAND_ARGUMENT> Both COMMAND_ID and COMMAND_ARGUMENT are integers.

Example usage:

int command_id;
int command_arg;
int status = ParseCommand("<10:42>", &command_id, &command_arg);
// status == 2 (both values parsed)
